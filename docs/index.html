Agentic Rubric Runner

Auditable Function-Calling Agent Workflow for PDF Generation and Rubric Evaluation

GitHub 仓库 Streamlit Demo

项目简介

agentic-rubric-runner 是一个可安装、可审计的文档约束型 Agent 流水线。 输入任务说明与 PDF 附件后，Agent 通过 Function Calling 自主读取文件、生成结构化报告并渲染为 PDF； 随后 Evaluator 依据 Rubric 标准自动评分，输出 JSON 结果与完整工具调用轨迹。

适用于尽职调查审核、合规检查、提案评审等「参考文档 + 评分标准 → 生成产物并打分」场景。

推荐安装（从 GitHub）

pip install "agentic-rubric-runner[web] @ git+https://github.com/bosprimigenious/agentic-rubric-runner.git"

export DEEPSEEK_API_KEY="sk-your-key"
agentic-rubric --help
agentic-rubric ui

带 [web] 会一并安装 Streamlit，否则 agentic-rubric ui 可能报错。

pipx 全局安装

pip install pipx && pipx ensurepath

pipx install "agentic-rubric-runner[web] @ git+https://github.com/bosprimigenious/agentic-rubric-runner.git"

安装指定版本

pip install "agentic-rubric-runner[web] @ git+https://github.com/bosprimigenious/agentic-rubric-runner.git@v0.4.0a1"

开发安装

git clone https://github.com/bosprimigenious/agentic-rubric-runner.git
cd agentic-rubric-runner
pip install -e ".[dev,web]"
cp .env.example .env

CLI 运行示例

# 只跑 Phase 1
agentic-rubric phase1 \
  --query fixtures/query.txt \
  --pdf fixtures/attachment.pdf \
  --out outputs/demo

# 完整流水线
agentic-rubric run \
  --query fixtures/query.txt \
  --pdf fixtures/attachment.pdf \
  --rubrics fixtures/rubrics.json \
  --out outputs/demo

Web UI

Streamlit 界面支持上传 query.txt、attachment.pdf、rubrics.json，分步执行 Phase 1 与 Phase 2：

  • Phase 1 — Agent 生成 PDF、Markdown 与 agent_trace.jsonl
  • Phase 2 — Rubric 自动评分，下载 grading_result.json
streamlit run app.py
# 或
agentic-rubric ui

在线 Demo：agentic-rubric-runner.streamlit.app（公开访问，需自备 DeepSeek API Key）

技术栈

  • Python 3.10+
  • OpenAI SDK (DeepSeek)
  • Function Calling
  • PyMuPDF
  • ReportLab
  • Pydantic v2
  • Typer + Rich
  • Streamlit

审计特性

  • agent_trace.jsonl — 每步工具调用的 step / timestamp / duration_ms
  • Phase 1 路径白名单 — 禁止读取 rubrics.json
  • run_meta.json — run_id 与输入文件 SHA256
  • 结构化错误码 E001–E006
  • 评分结果 Pydantic 校验 + 分数强制重算
